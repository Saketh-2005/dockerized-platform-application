# Dockerized DevOps Monitoring Dashboard

- Developed a Dockerized Node.js application using Express.js for deployment-ready web services.
- Designed a modern DevOps dashboard with real-time system information and health monitoring.
- Containerized the application using Docker for portability and isolated runtime environments.
- Implemented REST API endpoints for dashboard data and application health checks.
- Deployed the application on AWS EC2 using Docker containers.
- Configured Linux-based environments for application hosting and management.
- Utilized Git and GitHub for version control and collaboration.
- Demonstrated cloud deployment, containerization, and DevOps best practices.
- Technologies: Node.js, Express.js, Docker, AWS EC2, Linux, Git, GitHub.
